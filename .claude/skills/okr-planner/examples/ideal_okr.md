# Эталонный OKR — Q3 2026 (аннотированный)

> Этот документ — образец артефакта, который генерирует `/okr-deliver`. Аннотации `<!-- -->` — пояснения для LLM, в финальный документ не включаются.
>
> **Предметная область примера — иллюстративная** (data-платформа / билетный продукт). Доменные термины (Legacy-платформа, Шина, Партнёр-API, имена стейкхолдеров) — образец заполнения, не часть фреймворка. Для своего проекта замените их на свои из `.claude/domain-profile.md`. Универсальна именно **структура**: колонки таблицы, теги, IMP-шкала, БЫЛО→СТАЛО, прогресс-бар.

---

Проектные цели:
- **OBJ 1**: Отделение витрины Афиша от Legacy-платформы (виджеты&контент)
- **OBJ 2**: Развитие витрины Афиша — продажи в обход legacy_db
- **OBJ 3**: ACTIVITY поддержка текущих систем (API-слой, Процессинг, ext-api)
- **OBJ 4**: ACTIVITY: Закрытие Процессинг витрины
