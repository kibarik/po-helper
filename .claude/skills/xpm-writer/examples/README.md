# Примеры карт процесса-опыта (golden references)

Эталонные карты для калибровки стиля и проверки «на что похоже хорошо». Автор метода и карт — Андрей Шапиро / Byndyusoft (источники под AGPL-3.0; здесь — ссылки, не копии).

## Разобранный пример (наш, для новичка)
- **Доставка пиццы** — сквозной разбор всех 5 шагов: `docs/XPM/primer/skvoznoy-primer.md`.

## 9 эталонных карт автора (PDF в базе знаний)
Репозиторий `Byndyusoft/xp-mapping`, папка `xpm-examples/`:

| # | Карта | Чем полезна как образец |
|---|---|---|
| 1 | Сервис покупки рекламы на цифровых экранах | сложная услуга, много агентов |
| 2 | Услуга печати в копи-центре | компактная карта опыта |
| 3 | Опыт покупки автозапчастей | ветвления, каналы |
| 4 | Процесс составления карты процесса-опыта (XPM про XPM) | саморефлексивный образец нотации |
| 5 | Обобщённая услуга | шаблон «всех элементов» |
| 6 | Путь ассимиляции метода | личная траектория |
| 7 | Процессы ценообразования | подсистема, аннотации |
| 8 | Процесс деплоя пакетов (DevOps) | технический процесс без потребителя |
| 9 | Цикл договорённости «пинг-понг-понг» | петли взаимодействия |

Прямые ссылки: [github.com/Byndyusoft/xp-mapping/tree/main/xpm-examples](https://github.com/Byndyusoft/xp-mapping/tree/main/xpm-examples).

## Шаблоны для рисования
Figma (v3 автолэйаут), Holst.so, Miro, Excalidraw-библиотека `xpm-library.excalidrawlib`, OmniGraffle-стенсил — в `Byndyusoft/xp-mapping/templates`. Официальный инструмент метода — Социотех (`sociotech.center`).

> При калибровке стиля вывода: рендер должен быть лаконичным и читаемым без знания нотации — как эталоны выше. Меньше значков, больше смысла.
