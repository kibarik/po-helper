# Цветовые коды Карты гипотез

Цвета — **не для красоты**, а чтобы любой человек мгновенно читал чужую карту и карты можно было соединять между собой. Кодировка **единая для всех карт** — держись её. Автор метода — Александр Бындю (byndyu.ru).

> Источник: `docs/HYPOTHESIS-MAP/osnovy/elementy.md` (карточки) и `.../postroenie/shag-5-prioritizaciya.md` (приоритет стрелок).

---

## Карточки элементов

| Элемент | Цвет | HEX |
|---|---|---|
| 🟩 Цель, метрики | салатовый | `#cadf58` |
| 🟨 Субъект, мотивация | жёлтый | `#ffc831` |
| 🟧 Негативный субъект | оранжевый | `#ffb373` |
| 🟡 Гипотеза | светло-жёлтый | `#ffef73` |
| 🟦 Задача | голубой | `#a6cdff` |
| ⬜ Заметка | серый | `#f1f1f1` |
| 🟥 Блокер | красно-розовый | `#ffa391` |

> ⚠️ **Не перекрашивай карточки** под свои нужды (напр. задачи по направлениям): сломается кодировка. Группируй **подложками, колонками или тэгами**, а цвет карточки оставляй по типу элемента.

---

## Подложки групп

При топологиях с группировкой (доска задач, разбивка по направлениям, `resources/topologies.md`) используют **полупрозрачные зоны-подложки** под карточками — по отделу/направлению/статусу. Подложка — нейтральный фон (`opacity ~30`), карточки внутри сохраняют свой цвет по типу. Подложка группирует, карточка кодирует тип — два независимых слоя.

## Стрелки приоритета

Приоритет ставится **на стрелку**, выходящую из элемента (не на карточку). Складывается из двух оценок: 💥 **вес влияния** на цель × 💰 **стоимость проверки**. Самые вкусные — большое влияние + низкая стоимость.

| Приоритет | Цвет стрелки | Толщина | Excalidraw |
|---|---|---|---|
| 🔴 Высокий | красная | толстая | `strokeColor:"#e03131"`, `strokeWidth:4` |
| 🟡 Средний | жёлтая/оранжевая | средняя | `strokeColor:"#f08c00"`, `strokeWidth:2.5` |
| 🟢 Низкий | зелёная | тонкая | `strokeColor:"#2f9e44"`, `strokeWidth:1` |

Так на карте проступает **приоритетный путь** — заметная «красная дорожка» из самых важных веток; по ней идут первыми.

> Приоритет **не навсегда** — пересматривается на страткаденциях. Цвет стрелки ≠ цвет карточки: карточка гипотезы остаётся жёлтой, меняется только стрелка. Приоритет можно ставить любому элементу, из которого выходит стрелка (гипотезе, субъекту, метрике, задаче).

Нейтральная связь без приоритета — тонкая тёмно-серая (`strokeColor:"#495057"`, `strokeWidth:1.5`).
