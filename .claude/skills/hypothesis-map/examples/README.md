# Примеры карт гипотез (golden references)

Эталонные карты для калибровки стиля и проверки «на что похоже хорошо». Автор метода — **Александр Бындю** (byndyu.ru); база знаний под лицензией AGPL-3.0 — здесь ссылки, не копии. Для коммерческого использования метода — к автору.

## Разобранный пример (наш, для новичка)
- **«Высыпаться»** — сквозной разбор всех 5 шагов (цель → субъект → гипотезы → задачи → приоритизация) с ASCII-картой: `docs/HYPOTHESIS-MAP/primer/skvoznoy-primer.md`. Он же — источник рабочего JSON в `resources/render_excalidraw.md`.

## Учебные примеры внутри метода
| Пример | Где | Чем полезен |
|---|---|---|
| Умная лампочка (бизнес) | `docs/HYPOTHESIS-MAP/osnovy/format-gipotezy.md` | эталон гипотезы через субъект-покупателя |
| Автоматизация выписки счетов | `.../osnovy/format-gipotezy.md` | эталон гипотезы **без субъекта** (напрямую к цели) |
| Завтраки в кафе (малый бизнес) | `.../osnovy/format-gipotezy.md` | связка боль субъекта → идея → метрика |
| Автосервис (мастер-приёмщик) | `.../postroenie/shag-2-subekty.md` | глубокая мотивация субъекта, «5 раз ну и что?» |
| «Как не надо → как надо» ×11 | `.../kachestvo/tipovye-oshibki.md` | антипаттерны под каждый hard-гейт |

## Шаблоны и инструменты для рисования
- **Официальные с нотацией метода:** Социотех (`sociotech.center`), Гипотезная (`maps.integrostrat.ru`).
- **Онлайн-доски по шаблону:** Холст, Miro, Эсборд, Unidraw.
- **Программный рендер:** Excalidraw MCP — рецепт и рабочий JSON в `resources/render_excalidraw.md`.
- Бумажные стикеры не рекомендуются: карту постоянно перестраивают, а их тяжело двигать и соединять.

> При калибровке стиля вывода: карта должна читаться слева направо без знания нотации — как разбор «Высыпаться». Меньше значков, больше прослеживаемых связей от цели до задачи.
