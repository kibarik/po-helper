# Три рабочие области (справка для `/swot-factors`)

Операционная карта поля анализа: где что лежит, какие вопросы задавать, каким шаблоном формулировать карточку, каким цветом красить. Автор метода — **Александр Бындю** ([byndyu.ru](https://byndyu.ru)). Пересказ своими словами; человеческая версия с бытовыми примерами — `docs/LINKED-SWOT/osnovy/oblasti.md`.

---

## Раскладка поля

```
┌──────────────────┬────────────────────────┬──────────────────┐
│ ВНУТРЕННИЕ        │   СТРАТЕГИЧЕСКИЕ        │   ВНЕШНИЕ         │
│ ФАКТОРЫ           │   СТАВКИ                │   ФАКТОРЫ         │
│ (про нас)        │   (что делать)         │   (про мир)      │
├──────────────────┤                        ├──────────────────┤
│ 🟢 Сильные        │   🟡 идеи-ставки        │   🟣 Тренды       │
│    (сверху)      │      со связями        │    (сверху)     │
│                  │      влево и вправо    │                  │
│ 🔴 Слабые         │                        │   🟠 Угрозы       │
│    (снизу)       │                        │    (снизу)      │
└──────────────────┴────────────────────────┴──────────────────┘
```

**Главный водораздел:** «это про меня/нас или про мир?». Про нас (умения, ресурсы, привычки) → **внутреннее** (слева). Про мир (рынок, мода, законы, конкуренты) → **внешнее** (справа). Путаница внутреннее↔внешнее — самая частая ошибка (см. `hard_gates.md`, гейты G1–G3).

Каждая карточка получает **ИД** внутри своей области (С1, С2 / Сл1, Сл2 / Т1, Т2 / У1, У2), чтобы на неё ссылаться из ставок. Приоритет — **положением**: выше в колонке = приоритетнее (см. `bets.md` и `postroenie/README.md`).

## 🟢 Сильные стороны (внутренние, `#AFE192`)

Что вы или компания делаете хорошо. Козыри, уникальные ресурсы.

| | |
|---|---|
| **Наводящие вопросы** | Что делаете хорошо? Каким уникальным ресурсом обладаете? За что вас хвалят, чем восхищаются? |
| **Шаблон** | *«Мы устроены так, что … — и это нам благоволит».* |
| **Примеры** | Сильная экспертиза в психотерапии · приятная внешность и обаяние · свои лоббисты в нужных кабинетах · раскрученный бренд работодателя. |
| **Проверка** | подставь «я/мы» в начало — если разваливается («я — ИТ-инженеры хорошо зарабатывают»), это внешний атрибут, а не своя сторона. |

---

## 🔴 Слабые стороны (внутренние, `#FFACA7`)

Что хромает, чего не хватает, над чем работать.

| | |
|---|---|
| **Наводящие вопросы** | Что нужно улучшить? Какого ресурса не хватает? За что критикуют? |
| **Шаблон** | *«Мы сейчас организованы так, что … — и это нам мешает».* |
| **Примеры** | Слабая экспертиза в черновом ремонте · лишний вес · нет политического влияния · слабый бренд работодателя. |
| **Проверка** | недостаток профессии/группы («психологи мало зарабатывают») — это не ваша слабость; сделайте личным («я как психолог не умею на этом зарабатывать»). |

---

## 🟣 Тренды / возможности (внешние, `#BCC0F8`)

Куда движется мир вокруг. Ветер, который можно поймать в паруса.

| | |
|---|---|
| **Наводящие вопросы** | Куда «дует ветер»? Какие внешние факторы можно использовать? Какие возможности появились? Каким трендом воспользоваться? |
| **Шаблон** | *«Мир двигается в эту сторону, … — и это можно использовать».* |
| **Примеры** | Компании доплачивают за ИИ в продукте · девушкам стали нравиться умные парни · руководство компаний молодеет. |
| **Проверка** | начинается с глагола-действия («создать», «запустить») → это **ставка**, а не тренд. Тренд описывает движение мира, а не ваш поступок. |

> Частый капкан: «создать свой сайт и продавать с него» — это не тренд, а **следствие** тренда «покупатели ищут товар в поисковике, а не на маркетплейсе». Само действие всплывёт в центре как ставка.

## 🟠 Угрозы (внешние, `#FFB373`)

Что снаружи давит, пугает, грозит навредить. То, от чего защищаться.

| | |
|---|---|
| **Наводящие вопросы** | Какие факторы внешней среды несут угрозу? Какие изменения в отрасли беспокоят? |
| **Шаблон** | *«Ситуация давит на нас в этом, … — и это вынуждает строить защиту».* |
| **Примеры** | Компании урезают ИТ-бюджеты · могут отменить льготную ипотеку · слишком высокая ставка по кредитам. |
| **Проверка** | «это уже про меня и я это контролирую?» — если да, это слабая сторона (внутреннее), а не угроза. Угроза приходит снаружи и от нас не зависит. |

> Частый капкан: «слабо выстроены продажи» — это внутренняя слабость (в 🔴), а не угроза. Угроза: «конкуренты могут вложить большие деньги в отделы продаж».

---

## 🟡 Стратегические ставки (центр, `#FFEE73`)

Не фактор, а **действие**: «что нам с этим сделать». Формат `<действие>, чтобы <ценность>`, связи многие-ко-многим, баланс слева+справа. Полная механика — `bets.md`; человеческая версия — `docs/LINKED-SWOT/osnovy/stavki.md`.

## Цветовая шпаргалка

| Область | Карточка | Подложка зоны |
|---|---|---|
| 🟢 Сильные стороны | `#AFE192` | внутренние → `#F0F9F3` |
| 🔴 Слабые стороны | `#FFACA7` | внутренние → `#F0F9F3` |
| 🟡 Стратегические ставки | `#FFEE73` | ставки → `#FFFCEA` |
| 🟣 Тренды | `#BCC0F8` | внешние → `#F1F7FE` |
| 🟠 Угрозы | `#FFB373` | внешние → `#F1F7FE` |

Цвет — не украшение: по нему область читается без текста (зелёная — козырь, красная — боль, фиолетовая — попутный ветер, оранжевая — опасность). Раскладка в рендере — `render_excalidraw.md`.

## Мини-чек «в правильной ли области карточка?»

| Спросите себя | Область |
|---|---|
| Про нас, зависит от нас, это плюс? | 🟢 Сильные стороны |
| Про нас, зависит от нас, это минус? | 🔴 Слабые стороны |
| Про мир, это попутный ветер? | 🟣 Тренды |
| Про мир, это опасность? | 🟠 Угрозы |
| Это «что нам сделать»? | 🟡 Ставка (не фактор!) |

> Принцип нулевого допуска: каждый фактор ← источник (интервью, наблюдение, исследование рынка/ЦА, отчёт, решение заказчика). Нет источника → `[УТОЧНИТЬ]`, дальше в ставки как факт не идёт (гейт G8 в `hard_gates.md`).
