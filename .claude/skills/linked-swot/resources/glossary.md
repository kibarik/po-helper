# Глоссарий Связанного SWOT

Термины метода (автор — **Александр Бындю**, [byndyu.ru](https://byndyu.ru)). Пересказ своими словами; человеческая версия с бытовыми примерами — `docs/LINKED-SWOT/osnovy/slovar.md`.

| Термин | Значение |
|---|---|
| **Связанный SWOT (Linked SWOT / ВВС-анализ)** | Усовершенствованный SWOT: в центре не 4 застывших квадрата, а открытое поле ставок; всё соединяется ссылками; приоритет — положением. ВВС = **В**нутренние, **В**нешние, **С**тратегические. |
| **Внутренние факторы** | Всё, что про нас и зависит от нас: умения, ресурсы, привычки, слабости. Левая колонка. Проверка: «это про меня?». |
| **Внешние факторы** | Всё, что про мир вокруг и от нас почти не зависит: рынок, мода, законы, конкуренты. Правая колонка. Проверка: «это про мир?». |
| **Сильная сторона** 🟢 | То, что делаем хорошо; козырь, ресурс. Зелёная карточка (`#AFE192`) сверху слева. Шаблон: «мы устроены так, что … — и это нам благоволит». |
| **Слабая сторона** 🔴 | То, что хромает; чего не хватает, за что критикуют. Красная карточка (`#FFACA7`) снизу слева. Шаблон: «мы организованы так, что … — и это нам мешает». |
| **Тренд (возможность)** 🟣 | Куда движется мир; попутный ветер, который можно поймать. Фиолетовая карточка (`#BCC0F8`) сверху справа. Факт о мире, **не** ваша идея, что с ним делать. |
| **Угроза** 🟠 | Что снаружи давит и грозит навредить; опасность, от которой защищаются. Оранжевая карточка (`#FFB373`) снизу справа. Внешняя опасность, **не** внутренняя слабость. |
| **Стратегическая ставка** 🟡 | Идея-ход, превращающая факторы в конкретный шаг развития. Формат `<действие>, чтобы <ценность>`. Жёлтая карточка (`#FFEE73`) в центре. То, ради чего весь анализ. |
| **Связь** | Ниточка от ставки к фактору, из которого она выросла. Внутренние крепятся слева (сильные сверху, слабые снизу), внешние — справа (тренды сверху, угрозы снизу). |
| **Многие-ко-многим** | Свойство связей: одна ставка опирается на много факторов, один фактор кормит много ставок. |
| **Баланс связок** | Правило: у ставки есть хотя бы одна связка слева (внутр.) и одна справа (внешн.). Соединяет ваши силы с движением мира. |
| **Отметка использования** | Маркер на факторе, который пошёл хотя бы в одну ставку. Подсвечивает задействованные и **простаивающие** («осиротевшие») факторы. |
| **Номер карточки (ИД)** | Уникальный номер карточки внутри её области (С1, Сл2, Т1, У2), чтобы удобно ссылаться — как номера страниц. |
| **Приоритизация положением** | Важность задаётся высотой: чем выше карточка в своей колонке, тем приоритетнее. Приоритеты не дублируются — у каждой своя высота. |
| **Приём «как раскроется в условиях»** | Способ родить ставку: «как то, что `<внутр. фактор>`, может раскрыться в условиях `<внешн. фактор>`?». Перебор пар внутр.×внешн. |
| **Волнообразный процесс** | Анализ собирают не за один заход, а волнами: набросали факторы → придумали ставки → вспомнили новые факторы → вернулись, дописали. |
| **Карта гипотез** | Соседний метод того же автора («Цель → Субъект → Гипотеза → Задача»), куда перетекают ставки. Стык — стадия `/swot-link`, навык `hypothesis-map`. |
| **Ставка → гипотеза** | Способ перехода: к ставке дописывают «то» (цель) и «потому что» (мотивация субъекта) — получается гипотеза Карты гипотез. |
| **Ставка → цель** | Способ перехода: крупная ставка (новый продукт) становится целью Карты гипотез с метриками и балансирующими метриками. |
| **Маятник** | Двусторонняя связь методов: застряли в стратегии → качнулись в анализ за топливом; набрали ставок → качнулись обратно в стратегию. |
| **`[УТОЧНИТЬ]`** | Пометка на карточке без источника. Такой фактор не идёт в ставки как факт (гейт G8). |
| **Светофор** | Итог валидации: 🟢 готово в Карту гипотез · 🟡 годно с оговорками · 🔴 возврат на стадию-источник. |
