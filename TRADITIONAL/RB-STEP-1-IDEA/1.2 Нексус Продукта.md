# 1.2 Нексус Продукта — Полная инструкция

## 🎯 Цель

Создать **Нексус Продукта** — понимание текущего состояния продукта и vision.

**Время:** 2-4 часа  
**После:** Твой Нексус Продукта v0.1 ready

---

## 📋 Что такое Нексус Продукта?

**Простыми словами:** "Справочник" о твоем продукте — текущем состоянии и будущем видении.

---

## 🎯 2 Сценария

### Scenario A: У тебя уже есть продукт
- Анализируешь current state
- Измеряешь gap → vision

### Scenario B: У тебя новый продукт (или идея)
- Создаешь vision
- Определяешь success metrics

---

## 📝 Phase 1: Current State Analysis (для существующих продуктов)

### Step 1.1: Feature Inventory
**Список всех фич:**
```
Active Features:
- Feature 1: [Name] — [Description] — [User adoption]
- Feature 2: [Name] — [Description] — [User adoption]
...
```

### Step 1.2: Metrics Dashboard
**Текущие метрики:**
```
Product Metrics:
- DAU/MAU: [X]
- Retention: [X%]
- Revenue: $[X]/month
- NPS: [X]
- Customer count: [X]
```

### Step 1.3: Customer Feedback Themes
**Main feedback:**
```
Positive:
- "Love [feature]"
- "Great [aspect]"

Negative:
- "Missing [feature]"
- "[Feature] is confusing"

Neutral:
- Requests for [improvement]
```

---

## 🎯 Phase 2: Vision Articulation (для новых продуктов)

### Step 2.1: Vision Statement Template

```markdown
## Product Vision

"For [target customer],
Who [statement of need],
The [product name] is a [product category],
That [key benefit/value proposition].

Unlike [competitors/alternatives],
We [primary differentiation/moat]."
```

**Пример:**
```
"For SMB marketing teams (1-10 people),
Who need rapid audience insights but can't afford enterprise research,
SparkToro is an AI market research platform,
That provides audience data in minutes, not weeks.

Unlike expensive agencies or DIY surveys,
We automate research with AI at 1/10th the cost."
```

### Step 2.2: Success Metrics Definition

```markdown
## Success Metrics

### mNSM (Monetizable North Star Metric)
- **Metric:** [Which metric drives revenue?]
- **Current:** [Current value]
- **Target (6 months):** [Goal]
- **Target (12 months):** [Goal]

### Leading Indicators
- Activation: [X% users complete onboarding]
- Engagement: [X% users use key feature weekly]
- Retention: [X% retain month-over-month]

### Lagging Indicators
- Revenue: $[X]/month
- Customer count: [X]
- Market share: [X%]
```

---

## 🎯 Phase 3: Gap Analysis

### Step 3.1: Define Current State → Vision Gap

```markdown
## Gap Analysis

### Current State
[Describe what product does today]

### Vision State
[Describe desired future state]

### Gap (What's missing)
- [Gap 1]: [Description] — [Priority]
- [Gap 2]: [Description] — [Priority]
- [Gap 3]: [Description] — [Priority]
```

**Пример:**
```
### Current State
Basic landing page + manual service (beta, 100 users)

### Vision State
AI-powered platform serving 10k SMBs with automated research

### Gap
1. AI automation (Priority: High)
2. Self-service onboarding (Priority: High)
3. Scalable infrastructure (Priority: Medium)
4. Advanced features (Priority: Low)
```

```markdown
# Нексус Продукта v0.1

## 1. Product Overview

## 3. Vision
[Product vision statement]

## 4. Success Metrics
- mNSM: [metric]
- Leading indicators: [list]
- Lagging indicators: [list]

## 5. Gap Analysis
- Current → Vision: [what's missing]

## Context Ripeness: [X]%
```
