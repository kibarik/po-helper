# 5.3 Обновление Нексусов

## 🎯 Цель

Зафиксировать все learnings в Нексусах (v1.0).

**Время:** 2-4 часа  
**После:** Все 3 Нексуса обновлены до v1.0

---

## 📝 Process

### Step 1: Update Нексус Рынка (1 hour)

**Add:**
```markdown
## Validation Insights

### What We Learned
- **Market need:** [What validated]
- **Segment fit:** [Which segments resonated]
- **Pricing feedback:** [Price sensitivity]

### Competitive Intelligence
- **Competitor responses:** [What competitors doing]
- **Market gaps:** [New opportunities discovered]

### Updated Context Ripeness
- **Previous:** [X]%
- **Current:** [Y]%
- **What improved:** [What added]
```

### Step 2: Update Нексус Продукта (1 hour)

**Add:**
```markdown
## Validation Results

### Customer Insights
- **Top quotes:** [Most impactful feedback]
- **Pain points validated:** [Which problems confirmed]
- **Unexpected learnings:** [Surprises]

### Feature Prioritization
- **Bunch decisions:** [Why these 5-7 features first]
- **Deferred features:** [What deferred, why]

### Product Vision Refinement
- **Original vision:** [Initial statement]
- **Refined vision:** [Updated based on learnings]
```

### Step 3: Update Нексус Роста (1 hour)

**Add:**
```markdown
## Validation Economics

### Unit Economics Updates
- **CAC updated:** [New estimate based on validation]
- **LTV updated:** [New estimate]
- **Payback updated:** [New estimate]

### Business Model Refinements
- **Pricing changes:** [Adjustments based on feedback]
- **Revenue model tweaks:** [Refinements]

### Growth Strategy
- **Acquisition insights:** [Which channels worked best]
- **Retention learnings:** [What drives retention]
```

### Step 4: Create v1.0 Summary (1 hour)

**Template:**
```markdown
# Нексусы v1.0 — Summary

## Context Ripeness Across All Нексусы

| Нексус | Previous | Current | Improvement |
|--------|----------|---------|-------------|
| Рынок | [X]% | [Y]% | [+Z]% |
| Продукт | [X]% | [Y]% | [+Z]% |
| Рост | [X]% | [Y]% | [+Z]% |
| **Average** | **[X]%** | **[Y]%** | **[+Z]%** |

## Key Learnings Summary

### What Validated
1. **[Learning 1]:** [Impact]
2. **[Learning 2]:** [Impact]

### What Changed
1. **[Change 1]:** [Why]
2. **[Change 2]:** [Why]

### What Surprised Us
1. **[Surprise 1]:** [How we adapted]
2. **[Surprise 2]:** [How we adapted]

## Bunch Rationale

**Why these 5-7 features:**
- [Feature 1]: [Rationale + CP]
- [Feature 2]: [Rationale + CP]

## Next Iteration
**When to update Нексусы:** [Timeline]
**What to track:** [Metrics to monitor]
```

---

## ✅ Output

**All 3 Нексуса updated to v1.0** + **Summary document**

## 🎯 Success Criteria

✅ All Нексусы ≥60% Context Ripeness  
✅ Validation learnings captured  
✅ Bunch rationale documented  
✅ Update schedule defined
