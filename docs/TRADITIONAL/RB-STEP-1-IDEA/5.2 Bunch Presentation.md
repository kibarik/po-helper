# 5.2 Bunch Presentation

## 🎯 Цель

Создать stakeholder-ready presentation.

**Время:** 2-4 часа  
**После:** Presentation deck + speaker notes

---

## 📋 Presentation Structure

### Slide 1: Title Slide
```
[BIG Idea Name]
[Tagline - one powerful sentence]

[Your name/team]
[Date]
```

### Slide 2: Problem Statement
```
## The Problem

[Who] has [problem]
when [context]
because [root cause].

**Impact:**
- [Statistic 1]
- [Statistic 2]
- [Customer quote]
```

### Slide 3: Solution
```
## Our Solution

[BIG Idea in 2-3 sentences]

**How it works:**
1. [Step 1]
2. [Step 2]
3. [Step 3]

**Key differentiator:** [What makes us unique]
```

### Slide 4: Market Opportunity
```
## Market Size

- **TAM:** $[X] млрд (total market)
- **SAM:** $[Y] млн (serviceable market)
- **SOM:** $[Z] тыс (obtainable market, 2 years)

**Market trends:**
- [Trend 1]
- [Trend 2]
```

### Slide 5: Target Segment
```
## Who We Serve

[Segment description]

**JTBD:**
- [Functional job]
- [Emotional job]

**Why this segment:**
- [Reason 1]
- [Reason 2]
```

### Slide 6: Business Model
```
## How We Make Money

**Revenue model:** [Subscription/Transaction/etc]

**Pricing:**
- [Tier 1]: $[X]
- [Tier 2]: $[Y]

**Unit economics:**
- CAC: $[X]
- LTV: $[Y]
- Payback: [X] months
```

### Slide 7: Competitive Advantage
```
## Why We Win

**Unlike competitors:**
- [Competitor 1]: [Their approach] → [Our advantage]
- [Competitor 2]: [Their approach] → [Our advantage]

**Our moat:**
- [Sustainable advantage 1]
- [Sustainable advantage 2]
```

### Slide 8: Validation Results
```
## What We've Validated

✅ **[Hypothesis 1]:** [Method] → [Result]
✅ **[Hypothesis 2]:** [Method] → [Result]
✅ **[Hypothesis 3]:** [Method] → [Result]

**Average Confidence Point:** Increased from [X] to [Y]
```

### Slide 9: Bunch Overview
```
## What We'll Build First

**Top 5-7 features:**
1. **[Feature 1]** — CP: [X]/10 — [mNSM impact]
2. **[Feature 2]** — CP: [X]/10 — [mNSM impact]

**Bunch Window:** [X] months to business result
```

### Slide 10: Go-to-Market Plan
```
## How We'll Reach Customers

**Acquisition channels:**
1. [Channel 1]: [Est. leads/cost]
2. [Channel 2]: [Est. leads/cost]

**Launch timeline:**
- Month 0-2: [MVP build]
- Month 2-4: [Beta launch]
- Month 4-6: [Public launch]
```

### Slide 11: Financial Projections
```
## Financial Outlook

**Revenue projections:**
- Month 6: $[X]k MRR
- Month 12: $[Y]k MRR
- Month 18: $[Z]k MRR

**Key metrics:**
- Customers: [X] by month 12
- Gross margin: [X]%
- Runway: [X] months on current funding
```

### Slide 12: Resource Ask
```
## What We Need

**Team:**
- [Role 1]: [When needed]
- [Role 2]: [When needed]

**Budget:** $[X]k for [purpose]
**Timeline:** [X] months to [milestone]

**Expected outcome:** [What this will achieve]
```

### Slide 13: Risk Factors
```
## Risks & Mitigation

**Key risks:**
1. **[Risk 1]:** [Mitigation plan]
2. **[Risk 2]:** [Mitigation plan]
3. **[Risk 3]:** [Mitigation plan]
```

### Slide 14: Next Steps
```
## Next Steps

1. **Approve Bunch:** [Decision needed]
2. **Fund allocation:** [Budget approval]
3. **Team hiring:** [Roles to hire]
4. **Timeline:** [Start date, key milestones]

**Questions?**
```

---

## 📝 Speaker Notes

**Tips:**
- Tell a story, not just facts
- Use customer quotes
- Show validation evidence
- Be honest about risks
- Keep slides simple (<20 words each)

---

## ✅ Output

**Presentation deck** (14 slides) + **speaker notes**
