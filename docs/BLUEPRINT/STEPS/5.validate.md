# 5.validate — VALIDATE (Валидация: hard gates + Светофор)

> Стадия 5 · Роль: **Validator (свежий взгляд)** · Артефакт: `artefacts/validation.md`.
> Adversarial-проход отдельным запуском — как `/bft-validate`. Ломает confirmation bias автора диаграммы.

## 🎯 Цель

Проверить диаграмму по бинарным 🔴-гейтам ([[../principles]]) и выдать Светофор. Валидатор пытается **сломать** Blueprint, а не подтвердить.

## 👤 Роль

**Validator** — свежий взгляд, не участвовал в построении. Red-teamer: ищет висячие вертикали, скрытые fail-flows, уезд altitude.

## 📥 Вход

- `<service>.puml` (после WEAVE) + артефакты стадий 0–3.

## 🛠️ Hard-гейты (🔴 — бинарные)

| # | Гейт | 🔴 если |
|---|---|---|
| 1 | **Доказательная база** | есть блок без источника и без `[УТОЧНИТЬ]` |
| 2 | **As-Is первым** | To-Be нарисован, а As-Is отсутствует |
| 3 | **Вертикальная целостность** | есть висячая фронт-сцена (нет бэк-опоры) |
| 4 | **Нет сирот** | есть бэк-процесс без выхода во фронт |
| 5 | **Один актор** | в дорожках смешаны два клиента без разделения на ветки |
| 6 | **Altitude** | `Supporting Processes` ушли в API/схемы БД/sequence |
| 7 | **Negative flows** | значимые шаги без ветки-провала |
| 8 | **Моменты истины** | не отмечены, или отмечены над тонкой вертикалью без пометки риска |
| 9 | **Эмоции** | в `Customer Actions` нет эмоционального слоя |
| 10 | **Нет висячих стрелок** | стрелка упирается в пустоту |
| 11 | **Линии подписаны** | нет `Line of Interaction/Visibility/Internal Interaction` |
| 12 | **Границы дорожек** | действие лежит не в своей дорожке (бэк во фронте и т.п.) |

## 🚦 Светофор

- **🟢** — все гейты пройдены → в [[6.deliver]].
- **🟡** — пройдено, но есть `[УТОЧНИТЬ]` / открытые вопросы → отгрузка с явным списком рисков.
- **🔴** — нарушен хотя бы один инвариантный гейт (3/4/6/7/10) → **возврат на стадию-источник**:
  - вертикаль (гейты 3,4,10) → [[4.weave]] или [[3.backstage]];
  - altitude (6) → [[3.backstage]];
  - путь/эмоции/negative (7,9) → [[1.journey]];
  - актор/скоуп (5) → [[0.frame]].

## 📤 Выход — `validation.md`

Таблица гейтов (✓/🔴) + вердикт Светофора + список правок с адресом стадии.

## 🚫 Анти-паттерны

- «Замазать» дефект вместо возврата на стадию-источник.
- Валидировать той же ролью, что рисовала (нет свежего взгляда).
- Пропустить negative flows как «неглавное».
