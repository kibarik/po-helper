## 📊 Сводка бенчмарков (из исследования №4 и др.)

- **70%** микро-SaaS не преодолевают $500 MRR; **82%** не доходят до $5K MRR
- Медианный MRR микро-SaaS — **$500/мес**; только **18%** достигают $1K–5K MRR
- **43%** стартапов умирают из-за отсутствия PMF (CB Insights, 431 постмортем)
- Trial с картой → **48,8%** signup→paid; без карты — **18,2%**; freemium — **2,6%**
- 7-дневный trial превосходит 30-дневный на **71%** по конверсии
- Каждые 10 мин задержки TTV → **−8%** конверсии
- A/B: стандарт продуктовых команд — α=0.1, мощность 80%
- Win rate зрелых экспериментальных систем — **10–30%** (>50% = красный флаг)
- Компании, пересматривающие цены ежеквартально, растут на **30%** быстрее
