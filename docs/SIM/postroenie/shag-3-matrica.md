# 3️⃣ Шаг 3. Матрица — разложить истории в таблицу-карту

> Теперь превращаем набор историй в наглядную карту. У карты два направления, и оба несут смысл. Разберём, что куда кладётся.

---

## 🎯 Что делаем на этом шаге

Берём истории из [[shag-2-istorii|шага 2]] и раскладываем их в **матрицу** — таблицу. Матрица ориентирует истории **в двух направлениях сразу**, и каждое из них что-то означает.

---

## 🧭 Два направления карты

| Направление | Что означает | Простыми словами |
|---|---|---|
| **Слева → направо** | Следование процесса | «что за чем идёт» по времени |
| **Сверху → вниз** | Подчинение функции | «часть — целое»: крупная задача разбита на подзадачи |

### ➡️ Слева направо — ход дела во времени
Истории, которые идут одна за другой, ставим слева направо в порядке их выполнения. *Пример: «Выбор услуги» → «Выбор времени» → «Подтверждение».*

### ⬇️ Сверху вниз — от целого к частям
Если крупный шаг распадается на под-шаги, крупное ставим выше, а его части — ниже. *Пример: сверху «Оформление записи», а под ним — составляющие «Выбор услуги», «Выбор времени», «Подтверждение».*

> 💡 Проще запомнить так: **горизонталь — это «потом»**, а **вертикаль — это «состоит из»**.

---

## 🃏 Каждая история — это столбец из пяти карточек

Вот ключевой приём раскладки. Одну рабочую историю на карте изображают **столбиком из пяти карточек** — по одной на каждый элемент, сверху вниз в строгом порядке:

```
┌───────────────┐
│  Ц — цель      │  ← зачем во всём деле
├───────────────┤
│  Н — носитель  │  ← кто действует
├───────────────┤
│  С — ситуация  │  ← когда
├───────────────┤
│  Д — способ    │  ← как
├───────────────┤
│  О → О′        │  ← с чем работает → что получается
└───────────────┘
```

Сколько историй — столько столбцов рядом. Порядок карточек внутри столбца всегда одинаковый, поэтому карту легко «читать по строкам»: пробегитесь по строке носителей — увидите всех действующих; по строке ситуаций — все обстоятельства, и так далее.

---

## 🗺️ Как выглядит матрица целиком

Возьмём три истории из [[shag-2-istorii|шага 2]] и поставим их в ряд по времени. Каждая — столбец из пяти карточек.

```
        ← ход дела во времени (следование процесса) →

        История A          История B           История C
        Выбор услуги       Выбор времени       Подтверждение
      ┌────────────────┬───────────────────┬────────────────────┐
  Ц   │ не ошибиться   │ прийти в удобное  │ закрепить место,   │
      │ с выбором      │ время, без оче-   │ обе стороны знают  │
      │                │ реди              │                    │
      ├────────────────┼───────────────────┼────────────────────┤
  Н   │ Клиент         │ Клиент            │ Клиент             │
      ├────────────────┼───────────────────┼────────────────────┤
  С   │ решил постричь-│ услуга выбрана,   │ услуга и время     │
      │ ся, деталей нет│ видна занятость   │ выбраны            │
      ├────────────────┼───────────────────┼────────────────────┤
  Д   │ подбирает      │ выбирает свобод-  │ подтверждает и     │
      │ услугу         │ ное время        │ оставляет контакт  │
      ├────────────────┼───────────────────┼────────────────────┤
  О→О′│ «хочу постричь-│ список слотов →   │ черновик записи →  │
      │ ся» → услуга   │ выбранный слот    │ подтверждённая     │
      └────────────────┴───────────────────┴────────────────────┘
```

> 📌 Если бы «Оформление записи» было крупным шагом, а A/B/C — его частями, мы бы поставили карточку «Оформление записи» **выше** этих трёх столбцов — так на карте видно подчинение функции (целое над частями).

---

## 🧩 Зачем такая раскладка

- **Видно порядок.** Слева направо сразу читается, что за чем идёт.
- **Видно структуру.** Сверху вниз видно, из чего состоит крупная задача.
- **Ничего не забыто.** Пять строк-элементов заставляют для каждой истории заполнить всё — пустая клетка сразу бросается в глаза.
- **Готово место для реализации.** Под каждым столбцом останется место, куда на [[shag-4-formy-realizacii|шаге 4]] лягут формы реализации.

---

## 🚫 Частые промахи

- **Перепутали направления.** Под-шаги поставили в ряд (как будто по времени), хотя это «части целого» — им место друг под другом.
- **Сломали порядок карточек в столбце.** Держите единый порядок Ц-Н-С-Д-О во всех столбцах, иначе карту не прочитать по строкам.
- **Слишком много столбцов.** Если историй десятки — значит, дело слишком широкое. Вернитесь к [[shag-1-podgotovka|шагу 1]] и сузьте.
- **Сразу пишете под столбцами гаджеты.** Формы реализации — следующий шаг. Пока держим карту чистой.
